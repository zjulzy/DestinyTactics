# DestinyTactics
基于unity的简易战棋游戏demo
## 项目介绍
本项目是一个基于unity的简易战棋游戏demo，主要是为了学习unity和C#编程入门开发的，目前只有简单的战斗系统，后续可能会加入更多的内容。  
目前只支持玩家和AI对战，只有一个关卡  
## 项目实现
1. 游戏核心系统
游戏中的棋盘通过
2. 寻路算法
主要通过BFS和A*算法进行寻路，其中BFS负责寻找可移动范围和可攻击范围u，A*算法负责给出起点终点后寻找最短路径。
